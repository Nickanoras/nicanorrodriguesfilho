# Nicanor Rodrigues Filho — Site Pessoal

Site estático pronto para GitHub Pages.

## Como publicar em 5 passos

1. **Crie uma conta** em [github.com](https://github.com) (se ainda não tiver)

2. **Crie um repositório** com o nome exato:
   ```
   seuusuario.github.io
   ```
   Exemplo: se seu usuário for `nicanorrf`, o repositório se chama `nicanorrf.github.io`

3. **Suba o arquivo** `index.html` para esse repositório
   - Clique em "Add file" → "Upload files"
   - Arraste o `index.html`
   - Clique em "Commit changes"

4. **Ative o GitHub Pages**
   - Vá em Settings → Pages
   - Em "Source", selecione "Deploy from a branch"
   - Branch: `main` / pasta: `/ (root)`
   - Clique em Save

5. **Aguarde 1–2 minutos** e acesse:
   ```
   https://seuusuario.github.io
   ```

## Domínio próprio (opcional)

Se quiser usar um domínio como `nicanorrodrigues.com.br`:
- Compre o domínio (Registro.br, Locaweb, etc.)
- Em Settings → Pages → Custom domain, coloque seu domínio
- No painel do seu registrador, aponte os DNS para os IPs do GitHub Pages

## Atualizar o conteúdo

Para editar textos, abra o `index.html` no repositório → clique no lápis (Edit) → faça as alterações → Commit. O site atualiza em segundos.

## Adicionar posts reais

Substitua os posts de exemplo na seção `#blog` pelos seus títulos e links reais do WordPress.
